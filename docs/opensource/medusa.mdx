## Functionality

Medusa positions itself as a framework for code-level customization to achieve specific functionalities. For example, while the "Draft Order" feature can be adapted for use cases like Quotation, it doesn‚Äôt offer full functionality out-of-the-box. Instead, the Medusa team provides instructions (recipes) for extending features, such as B2B use cases.
