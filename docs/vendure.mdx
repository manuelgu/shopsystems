## Functionality

### Feature overview

Vendure is well-suited for B2C use cases and offers flexibility for basic Marketplace scenarios due to its ‚Äúmulti-everything‚Äù approach. However, it lacks an integrated CMS, so developers must integrate a custom solution.

Vendure offers unique enterprise features, including SSO integration with providers like Okta.

It does not come with a built-in search engine (e.g., Elasticsearch or Solr); instead, the Search & Filter functionality relies on the underlying relational database.

Although Vendure is open-source, certain official features require separate purchases. For instance, the ‚Äúwish list‚Äù feature costs ‚Ç¨480 per year. üåê

## Development approach

### Technology Stack

Vendure is fully built with TypeScript and utilizes the following technologies:

<div style={{ display: 'flex', justifyContent: 'space-around' }}>
    <div style={{ width: '50%' }}>
        - [**NestJS**](https://nestjs.com/) as the underlying Node.js framework
        - **TypeORM** for interacting with any database
        - **Angular** for rendering the Admin UI
        - **GraphQL** for API communication
